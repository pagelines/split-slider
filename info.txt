== Description ==
<h3>What is Split Slider?</h3>

Split Slider is a jQuery slider with a nice Splitting effect and a number of options including custom text fields, color control and more. Split Slider is beautiful responsive and supports up to 15 slides of your choice.

Want to start splitting? <strong>Grab it while it lasts! :)</strong>
